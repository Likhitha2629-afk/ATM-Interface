# 🏦 ATM Interface

A simple and interactive **ATM Interface** built using **HTML, CSS, and JavaScript**. This project simulates the basic functionalities of an Automated Teller Machine (ATM) through a user-friendly web interface.

---

## 📌 Project Overview

The ATM Interface allows users to log in securely and perform common banking operations such as checking their balance, depositing money, withdrawing funds, transferring money, viewing transaction history, and logging out.

---

## ✨ Features

* 🔐 Secure Login (User ID & PIN)
* 💰 Balance Inquiry
* 💵 Deposit Money
* 🏧 Withdraw Money
* 🔄 Transfer Money
* 📜 Transaction History
* 🚪 Logout Functionality
* 📱 Responsive User Interface

---
